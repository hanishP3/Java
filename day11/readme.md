Step 1: The "Universal Box" (Generics Practice)
First, let's build the container. We want a class that can hold any data type.

The Task:

Create a class DataJar<T>.

Give it a private variable T content.

Create a constructor that takes T as an argument.

Create a method public T getContent() to return the value.

In your Main:

Create a DataJar for a String ("Secret Message").

Create a DataJar for an Integer (12345).

Print both using the getContent() method.
