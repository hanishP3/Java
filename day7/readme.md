The "One & Done" Challenge for Today
Since you're feeling light, just do this one exercise to lock it in:

Create an abstract class Appliance.

Add a non-abstract method void turnOn() that prints "Powering up...".

Add an abstract void work(); method.

Create a class Fan that extends Appliance.

In the Fan class, implement work() to print "Spinning blades...".

In main, try to create an object of Appliance (Watch it fail!), then create a Fan and call both methods.
