The Smart Thermostat Task
Variables: Create an int temp (set it to 25) and a String status.

The Logic: * If temp is above 30, status is "HOT".

If temp is between 15 and 30 (inclusive), status is "WARM".

Otherwise, status is "COLD".

The Comparison: * Create a second String userSetting set to "WARM".

Write an if statement to check if status matches userSetting. If they match, print "System Optimized".
