The Challenge:

Create a class SmartBulb.

Add a private int brightness; (0 to 100).

Create a Getter getBrightness().

Create a Setter setBrightness(int value).

Logic: If the value is between 0 and 100, update the variable.

Logic: If the value is < 0, set it to 0. If > 100, set it to 100.

Logic: Print a message saying what happened.

Why this matters: This is exactly how you'll handle data in your E-commerce course (Phase 1). You never just "trust" the user's input; you validate it in the Setter.
