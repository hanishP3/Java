The "Data Cleaner" Challenge
Scenario: You received a messy username from a database: "  User_Hanish_99  ".
Before you can use it in your app, you need to clean it up and extract specific parts.

Your Task:

Clean the Space: Create a String that removes the leading and trailing spaces. (Hint: Use the .trim() method).

Shout It: Convert the cleaned name to all Uppercase.

The Sniper: Print the last character of the name using .charAt() and .length(). (Don't hardcode the number—use math so it works for any name!).

The Slice: Use .substring() to extract just the middle part: "HANISH".
